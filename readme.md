 # **E-Commerce Application Backend**


## Live link: https://e-commerce-backend-iota-pied.vercel.app

## Project Description

This project is an E-Commerce application backend built with Node.js and Express.js. It provides a RESTful API for managing products, users, orders, and payments. The application uses Prisma ORM to interact with a PostgreSQL database and is written in TypeScript for type safety and better code maintainability. The backend is designed to be scalable, secure, and easy to maintain, making it a solid foundation for any e-commerce platform.

## Technology Stack & Packages
  - Node.js
  - Express.js
  - Prisma ORM
  - PostgreSQL
  - Typescript
  - cors

# How to run the application locally

1. Clone the repository

```
https://github.com/Khairul-islam98/e-commerce-backend.git
```

2. Project open

```
cd e-commerce-backend
```

3. install the required packages

```
npm i
```

4. Add a .env file

```
PORT=5000
DATABASE_URL
```

5. Run the Application locally
